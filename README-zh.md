# L10n

[English](/README.md) | 中文

MrXiaoM 插件的本地化配置文件

## 如何使用

克隆这个仓库，然后按以下文件夹格式，寻找你想要的插件和语言。

```
- mcio-dev/localization
  |- 插件名
     |- XX-XX (语言代码)
        |- config.yml
        |- messages.yml
```

举个例子，我是说英语的，我需要 [SweetMail](https://github.com/MrXiaoM/SweetMail) 的语言文件，所以我需要查看 `SweetMail/en-us/` 里面的文件。
